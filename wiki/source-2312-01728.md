# ImputeFormer 论文摘要

**ImputeFormer** (Low Rankness-Induced Transformers for Generalizable Spatiotemporal Imputation) 是由同济大学和香港理工大学的研究者于 KDD 2024 发表的时空数据填补模型。论文首次将低秩归纳偏置系统性地引入 Transformer 架构，实现信号-噪声平衡的通用时空填补[^src-2312-01728]。

## 核心贡献

1. **低秩性引导的 Transformer**：首次将低秩归纳偏置（low-rankness inductive bias）引入时空 Transformer 架构，在强归纳偏置和高模型表达力之间取得平衡，实现信号与噪声的平衡表征。

2. **投影注意力机制**（Projected Attention）：在时间维度上通过可学习投影器将注意力矩阵分解为低秩形式，实现线性复杂度 O(TC)，并显式抑制高秩虚假相关性。

3. **嵌入注意力机制**（Embedded Attention）：在空间维度上利用节点嵌入的稠密抽象作为低维代理计算相关性图，避免在全量数据上计算全局注意力，复杂度 O(N·D_emb) 线性于传感器数量。

4. **傅里叶稀疏性损失**（Fourier Imputation Loss, FIL）：基于卷积核范数与傅里叶 ℓ1 范数的等价性，在频域对填补结果的频谱进行稀疏正则化，促进低秩结构。

## 核心动机（频谱视角）

论文明确揭示了低秩模型和深度模型在奇异值频谱上的互补偏置（图 1）：

- **低秩模型**：截断频谱左侧过多能量 → 过度平滑填补
- **深度模型**：保留频谱右侧过多能量 → 保留噪声的尖锐重构
- **ImputeFormer**：通过低秩性约束在两者间取得平衡 → 填补结果的奇异值分布最接近完整数据分布

## 核心方法

### 架构概览

```
输入嵌入层 → L× 时空交互层 → MLP 读出
                 ├── 时间投影注意力 (线性复杂度 O(TC))
                 └── 空间嵌入注意力 (线性复杂度 O(N·D_emb))
```

### 关键技术

- **输入嵌入**：通过在额外维度上展开稀疏观测并投影为隐藏状态，保留不完整时间序列的信息密度。结合时间戳编码（正弦-余弦位置编码表示一天内时间位置）和可学习节点嵌入。

- **时间投影注意力**：两层注意流程——先通过可学习投影器 P_proj ∈ R^(C×D') 将输入压缩为紧凑表示（Inflow），再用投影器作为键字典恢复完整序列（Outflow），等价于对注意力矩阵做显式低秩分解：Attention ≈ Q(P^T P)K^T，秩 r ≤ min(C, D')。

- **空间嵌入注意力**：通过节点嵌入的线性投影计算传感器间相关性，利用矩阵乘法的结合律和分离 softmax 实现线性复杂度 O(N·D_emb)。生成的注意力图秩 r ≤ min(N, D_emb)。

- **傅里叶填补损失（FIL）**：DFT 对角化循环矩阵 ⇒ ‖DFT(x)‖_1 = ‖C(x)‖_*（核范数），从而用 FFT 高效优化频谱稀疏性。总损失 L = L_recon + λ·L_FIL。

## 实验结果

在 10 个基准数据集（METR-LA、PEMS-BAY、PEMS03/04/07/08、SOLAR、CER-EN、AQI、AQI36）上，ImputeFormer 在点缺失（Point missing）和块缺失（Block missing）两种模式下均取得 SOTA。点缺失指随机位置独立缺失（更常见），块缺失指连续时间段整段缺失（更具挑战性）[^src-2312-01728]。关键指标：

| 数据集 | 场景 | 相比 SPIN (MAE) |
|--------|------|------------------|
| METR-LA | Point 12.5% | 5.8% ↓ |
| PEMS-BAY | Point 12.5% | 5.9% ↓ |
| PEMS08 | Point 12.5% | 7.5% ↓ |
| PEMS08 | Block 12.5% | 21.3% ↓ |
| SOLAR | Point | 15.9% ↓ |
| AQI | Point | 13.2% ↓ |

块缺失场景下 ImputeFormer 的优势更为显著（如 PEMS08 Block +21.3%），说明全局时空建模相比局部依赖建模在连续缺失上更具优势。

### 计算效率

相比 SPIN 训练速度提升约 **15×**（图 3），且对输入序列长度呈线性扩展（图 4），长序列下效率优势进一步放大。

### 鲁棒性

- **测试阶段**：观测率从 12.5% 降至 1% 时，ImputeFormer MAE 增幅最小，始终 rank 1
- **训练阶段**：仅用 1% 训练观测率时，在密集测试场景下仍保持竞争力，低秩先验对极度稀疏场景有效约束
- **零样本迁移**：训练数据与测试数据分布不一致时仍表现稳定

### 消融实验

| 变体 | PEMS08 Point MAE | PEMS08 Block MAE |
|------|:-----------------:|:-----------------:|
| 完整 ImputeFormer | **11.01** | **12.50** |
| − 移除 FIL | 11.35 (+3.1%) | 13.07 (+4.6%) |
| − 移除投影注意力 | 11.48 (+4.3%) | 13.57 (+8.6%) |
| − 移除嵌入注意力 | 11.54 (+4.8%) | 13.36 (+6.9%) |
| − 移除 FIL + 层次化损失 | 11.78 (+7.0%) | 13.89 (+11.1%) |

空间嵌入注意力在 Block 场景下更重要（+6.9%），说明块缺失需要全局空间依赖。层次化损失替代 FIL 后性能下降最显著（+11.1%），证明 FIL 的简单且有效。

## 可解释性分析

- 奇异值频谱分析确认 ImputeFormer 的填补结果在信号-噪声之间取得最佳平衡
- 节点嵌入的 t-SNE 可视化显示临近传感器形成聚类
- 投影注意力的 inflow/outflow 可视化揭示信息压缩与恢复过程
- 消融实验验证时间/空间模块各自的重要性，FIL 在未观测点上提供有效正则化

## 局限性

- 时间维度的投影注意力假设序列存在冗余性（低秩性），在极短序列上可能受限
- 空间嵌入注意力依赖于节点嵌入的质量，在传感器数量极少时效果减弱
