# Wiki Template — Reading Log

> 用于书籍、文章的阅读记录。

## 项目背景

- **主题**: 阅读日志
- **目的**: 书籍、文章、通讯摘要与交叉引用
- **主要语言**: zh

## 建议 wiki/ 结构

```
wiki/
  sources/        # 书籍/文章摘要 (source-summary)
  authors/        # 作者 (entity)
  ideas/          # 核心观点 (concept)
  quotes/         # 引语集
  reviews/        # 书评 (analysis)
```

## 额外 Frontmatter 字段 (建议)

- `rating: 1~5` — reviews 页面
- `finished: YYYY-MM-DD` — sources 页面
- `genre: fiction | non-fiction | essay` 等
