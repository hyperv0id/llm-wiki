# Wiki Template — Personal Notes

> 개인 메모·일지·학습 노트용.

## 프로젝트 맥락

- **주제**: 개인 노트
- **목적**: 일기·학습 메모·인사이트 축적
- **주 언어**: ko

## 권장 wiki/ 구조

```
wiki/
  daily/          # 날짜별 (YYYY-MM-DD.md)
  topics/         # 주제별 장기 메모
  people/         # 사람
  projects/       # 개인 프로젝트 기록
```

## 특이 사항

- Citation 의무는 완화 (source가 "나"인 경우가 많음)
- 대신 `confidence: low` 기본값 권장 — 추후 외부 소스로 보강 시 갱신
