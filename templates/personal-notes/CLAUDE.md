# Wiki Template — Personal Notes

> 用于个人备忘、日志、学习笔记。

## 项目背景

- **主题**: 个人笔记
- **目的**: 日记、学习备忘、洞察积累
- **主要语言**: zh

## 建议 wiki/ 结构

```
wiki/
  daily/          # 按日期 (YYYY-MM-DD.md)
  topics/         # 按主题的长期笔记
  people/         # 人物
  projects/       # 个人项目记录
```

## 特殊说明

- Citation 义务可放宽 (source 常为"我")
- 建议默认 `confidence: low` — 后续用外部来源补充时更新
